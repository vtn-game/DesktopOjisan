# UXデザイン

## コンセプト

**「チルウィズユー」の完全逆張り**

癒し系デスクトップマスコットの対極として、うざいけど憎めないおじさんキャラクターを配置する。

### 参考作品

- [Chill with You : Lo-Fi Story](https://store.steampowered.com/app/3548580/Chill_with_You__LoFi_Story/) - 癒し系マスコット
- デスクトップおじさん - **逆張り：うざいけど役に立つおじさん**

## ターゲットユーザー

### プライマリ

- エンジニア・プログラマー
- 在宅ワーカー
- 長時間PC作業者

### ペルソナ

```
名前: 田中太郎（仮）
年齢: 28歳
職業: Webエンジニア
課題:
  - 集中力が続かない
  - サボりがち
  - 誰かに見られていないとダラける
ニーズ:
  - 適度なプレッシャー
  - 作業の区切り
  - ちょっとした息抜き
```

## 体験設計

### コア体験

1. **監視されている感覚** - おじさんが見ている
2. **適度な緊張感** - サボると怒られる
3. **達成感** - 褒められると嬉しい
4. **愛着** - だんだんおじさんが好きになる

### エモーショナルジャーニー

```
初回起動
    │
    ▼ 「誰だこのおじさん」
最初の怒り
    │
    ▼ 「うざっ」
最初の褒め
    │
    ▼ 「あ、ちょっと嬉しい」
継続利用
    │
    ▼ 「なんか居ないと寂しい」
愛着形成
    │
    ▼ 「おじさんがいるから頑張れる」
```

## UIデザイン方針

### 視覚的特徴

| 要素 | 方針 |
|------|------|
| おじさん | 目立つが邪魔にならない位置 |
| ダイアログ | 読みやすく、すぐ消える |
| ボタン | 小さめ、控えめ |
| 全体 | 作業を邪魔しない |

### 色彩

- メイン: 落ち着いた色調
- アクセント: おじさんの感情を表す色
  - 怒り: 赤系
  - 喜び: 黄系
  - 通常: グレー系

### レイアウト

```
┌────────────────────────────────────┐
│                                    │
│          ユーザー作業領域          │
│                                    │
│                                    │
│                                    │
│                       ┌──────────┐ │
│                       │ おじさん │ │
│ ┌───────────────────┐ └──────────┘ │
│ │   ダイアログ      │   [コマンド] │
│ └───────────────────┘              │
└────────────────────────────────────┘
```

## インタラクションデザイン

### 原則

1. **非侵襲的** - 作業を中断させない
2. **予測可能** - 何が起きるか分かる
3. **制御可能** - いつでも消せる
4. **適度な頻度** - うるさすぎない

### フィードバック

| アクション | フィードバック |
|-----------|---------------|
| ボタンホバー | カーソル変化 |
| ボタンクリック | 即座に反応 |
| メッセージ表示 | タイピングエフェクト |
| エラー | 控えめな通知 |
