# 一般ルール

## プロジェクト概要

- **名前**: デスクトップおじさん
- **ジャンル**: デスクトップマスコット
- **プラットフォーム**: Windows
- **エンジン**: Unity

## 開発方針

### 禁止事項

- キーボードロギング（マルウェア認定されるため）
- 過度なシステムリソース消費
- ユーザーの許可なしのネットワーク通信

### 推奨事項

- 常駐アプリとして軽量であること
- ユーザー操作を邪魔しないこと
- 簡単に終了できること

## ディレクトリ構成

```
DesktopOjisan/
├── Assets/
│   ├── Plugins/x86_64/     # NativePlugin DLL
│   ├── Prefabs/            # プレハブ
│   └── Scripts/
│       ├── Core/           # コアシステム
│       ├── Server/         # APIサーバ
│       ├── UI/             # UI関連
│       └── Editor/         # エディタ拡張
├── NativePlugin/           # C++ ネイティブプラグイン
├── spec/                   # 仕様書
└── Tools/                  # 開発ツール
```

## バージョン管理

- Git を使用
- main ブランチ: 安定版
- 機能ブランチ: feature/xxx

## ドキュメント

- 仕様書は `/spec` に配置
- 日本語で記述
