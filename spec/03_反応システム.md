# おじさん反応システム

## 怒る系

### Ctrl+V検知
- トリガー: 軽い気持ちでCtrl+Vする
- 反応: すげー怒る

### Steam長時間プレイ検知
- トリガー: Steamのゲームプレイ時間が3時間を超える
- 反応: 怒ってくる

### 長時間コミットなし
- トリガー: 長時間コミットしていない
- 反応: コミットしろと諭してくる

## 喜ぶ系

### コーヒー
- トリガー: コーヒーをあげる
- 反応: 喜ぶ

### コミットログ表示
- トリガー: コミットログを見せる
- 反応: 喜ぶ

### コミット検知
- トリガー: コミットする
- 反応: 褒めてくれる
- 条件: コードを書かないと褒めてくれない
