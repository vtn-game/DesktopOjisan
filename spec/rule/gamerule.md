# ゲームルール

## コンセプト

「チルウィズユー」の逆張り - うざいおじさんと仲良くなる

## ゲーム目標

- おじさんとコミュニケーションを取る
- おじさんの好感度を上げる（将来実装）
- 開発作業を続けながらおじさんと共存する

## 基本ルール

### おじさんの反応

おじさんは開発者の行動に反応する：

| 行動 | 反応 |
|------|------|
| Ctrl+V（コピペ） | 怒る |
| Steam 3時間以上 | 怒る |
| 長時間コミットなし | 説教 |
| コミット | 褒める |
| コーヒーをあげる | 喜ぶ |

### コマンドシステム

ユーザーはコマンドでおじさんと対話できる：

| コマンド | 効果 |
|---------|------|
| talk | おしゃべり |
| status | 状態確認 |
| help | ヘルプ表示 |
| exit | 終了 |

### 外部連携

APIサーバ経由で外部ツールと連携：

- VSCode拡張からコミット通知
- CLIツールからメッセージ送信
- Steam連携（将来）

## 禁止事項

以下の行動はゲームとして実装しない：

- キーボードロギング（マルウェア認定）
- ユーザーの作業を強制中断
- 閉じられないウィンドウ

## 将来構想

### 好感度システム

- 好感度に応じてセリフが変化
- 高好感度で特別なイベント
- 低好感度でおじさんが拗ねる

### スキン・カスタマイズ

- おじさんの見た目変更
- セリフパック追加
- ユーザー独自のリアクション設定
